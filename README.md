Trrendify – Clone d’Instagram
  Description du projet

Trendify est une application web inspirée d’Instagram, développée dans le cadre d’un projet scolaire.
L’application permet d’afficher un flux d’images provenant d’une API externe, de gérer des comptes utilisateurs et d’interagir avec le contenu via des publications, des likes et des commentaires.

Le projet respecte une architecture front-end / back-end séparée, avec une API REST développée en Node.js et Express.

   Objectifs pédagogiques

- Comprendre l’architecture d’une application web complète

- Manipuler une API REST

- Utiliser JavaScript côté client (vanilla)

- Gérer l’authentification des utilisateurs

- Mettre en pratique Git et le travail d’équipe

   Technologies utilisées
FRONT-END :

-HTML

-CSS

-JavaScript (vanilla)

-Fetch API

BACK-END:

-Node.js

-Express.js

-API REST

-Authentification (JWT ou sessions)



⚙️Fonctionnalités principales
FRONT-END :

-Interface responsive (style Instagram)

-Affichage d’un flux d’images

-Page de connexion et d’inscription

-Publication de messages

-Likes et commentaires

-Modèles et interactions dynamiques

BACK-END

-API REST (/api/auth, /api/posts, /api/likes, /api/comments)

-Authentification sécurisée

-Gestion des utilisateurs

-Protection des routes

INSTALLATION ET EXÉCUTION

1. Cloner le projet:
git clone https://github.com/votre-utilisateur/trendify.git

2️. Installer les dépendances du backend :
cd trendify-backend
npm install

3. Lancer le serveur backend :
npm start


Le serveur sera accessible à :

http://localhost:3000

4️. Lancer le front-end

Ouvrir simplement le fichier index.html dans un navigateur web.


Remarques importantes :

Le backend doit être lancé pour que le feed et la publication fonctionnent

Certaines routes sont protégées et nécessitent une connexion

L’API d’images utilisée respecte les conditions d’utilisation (attribution si requise)

Travail d’équipe
